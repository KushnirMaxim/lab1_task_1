HTML (Hypertext Markup Language), також відома в IT сфері як мова розмітки гіпертекстових документів, по факту є набором спеціальних елементів - тегів. Для того, щоб підключити CSS стиль до певного HTML тегу чи сукупності тегів використовуються спеціальні HTML атрибути class чи id. Наприклад, class="style_page" або id="torro-2".
